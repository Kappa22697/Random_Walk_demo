# ランダムウォーク シミュレーション

Random Walk Simulation with Configurable Probabilities

## 使い方

### 1. **通常モード（推奨）** - ファイルの先頭でパラメータを設定

`RW1.py` の先頭部分（12行目から）にある設定を編集してください：

```python
# ========================================
# パラメータ設定（ここを編集してください）
# ========================================

# シミュレーションの基本設定
SIMULATION_CONFIG = {
    'dimension': 2,        # 次元: 1 または 2
    'n_steps': 1000,       # ステップ数
    'n_walkers': 1,        # ウォーカーの数（1なら単独、複数なら集団シミュレーション）
}

# 1次元ランダムウォークの確率設定
CONFIG_1D = {
    'p_left': 0.3,         # 左に移動する確率
    'p_stay': 0.4,         # 停滞する確率
    'p_right': 0.3,        # 右に移動する確率
}

# 2次元ランダムウォークの設定
CONFIG_2D = {
    'mode': 'lattice',     # 'continuous' (連続角度) または 'lattice' (格子型)
    'p_stay': 0.1,         # 停滞する確率
    'p_right': 0.35,       # 右に移動する確率
    'p_up': 0.35,          # 上に移動する確率
    'p_left': 0.1,         # 左に移動する確率
    'p_down': 0.1,         # 下に移動する確率
}
```

設定を編集したら、実行：
```bash
python3 RW1.py
```

### 2. **対話モード** - ターミナルで質問に答える形式

```bash
python3 RW1.py interactive
```

質問に答えながらパラメータを設定します：
```
次元を選択 (1 or 2): 2
ステップ数 (例: 1000): 1000
移動モード (continuous/lattice, 例: continuous): lattice
右に移動する確率 (0-1, 例: 0.25): 0.35
上に移動する確率 (0-1, 例: 0.25): 0.35
...
```

### 3. **デモモード** - 様々なパターンを自動実行

```bash
python3 RW1.py demo
```

6つの異なる設定のランダムウォークが自動的に実行されます。

## パラメータ説明

### 基本設定
- **dimension**: 1（1次元）または 2（2次元）
- **n_steps**: シミュレーションのステップ数
- **n_walkers**: ウォーカーの数
  - `1`: 単独のランダムウォークを表示
  - `2以上`: 複数のウォーカーを同時にシミュレート

### 1次元ランダムウォーク
3つの確率の合計が **1.0** になる必要があります：
- **p_left**: 左（-1）に移動する確率
- **p_stay**: 停滞する確率（移動しない）
- **p_right**: 右（+1）に移動する確率

### 2次元ランダムウォーク

#### continuousモード（連続角度）
- 0°～360°の全方向にランダムに移動
- **p_stay**: 停滞する確率のみ設定

#### latticeモード（格子型）
5つの確率の合計が **1.0** になる必要があります：
- **p_right**: 右（→）に移動する確率
- **p_up**: 上（↑）に移動する確率
- **p_left**: 左（←）に移動する確率
- **p_down**: 下（↓）に移動する確率
- **p_stay**: 停滞する確率

## 例

### 例1: 右に偏った1次元ランダムウォーク
```python
SIMULATION_CONFIG = {
    'dimension': 1,
    'n_steps': 1000,
    'n_walkers': 1,
}

CONFIG_1D = {
    'p_left': 0.2,
    'p_stay': 0.3,
    'p_right': 0.5,  # 右に偏る
}
```

### 例2: 右上に漂流する2次元ランダムウォーク
```python
SIMULATION_CONFIG = {
    'dimension': 2,
    'n_steps': 2000,
    'n_walkers': 1,
}

CONFIG_2D = {
    'mode': 'lattice',
    'p_right': 0.35,  # 右へのドリフト
    'p_up': 0.35,     # 上へのドリフト
    'p_left': 0.1,
    'p_down': 0.1,
    'p_stay': 0.1,
}
```

### 例3: 複数のウォーカーで統計分析
```python
SIMULATION_CONFIG = {
    'dimension': 2,
    'n_steps': 500,
    'n_walkers': 50,  # 50個のウォーカー
}

CONFIG_2D = {
    'mode': 'continuous',
    'p_stay': 0.0,
}
```

## 理論的性質

- **対称ランダムウォーク**（偏りなし）の場合：
  - 原点からの期待距離: √n （nはステップ数）
  - 1次元と2次元では、必ず原点に戻る（再帰的）

- **非対称ランダムウォーク**（偏りあり）の場合：
  - ドリフト方向に漂流する
